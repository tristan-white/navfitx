# FAQs

#### Can I use this software to make my performance evaluation report?

It's important to note that as per BUPERSINST 1610.10H, service members are required to use NAVFIT98 to create their performance evaluation reports.

If the question is asking "is NAVFITX capable of creating PDF reports that are identical to PDF reports generated by NAVFIT98", the answer is "yes".

#### Why doesn't the NAVFITX GUI look more modern than NAVFIT98?

One complaint against eNavFit was that it was difficult to use and insufficient training was provided to learn how to use it; NAVFITX mimics the NAVFIT98 user interface so that NAVFIT98 users will find it intuitive to use. 

In the future, NAVFITX will also support a web interface so that reports can be generated by visiting a site in a browser. No software installation necessary.
